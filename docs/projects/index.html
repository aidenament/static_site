Welcome to my projects page! Here you'll find a collection of my work in mathematics, computer science, and machine learning. Each project represents a unique challenge I've tackled or random problem I decided to persue.

Project Directory

  • Sum of Uniform Random Variables - Finds the expected number of i.i.d variables uniformly distributed betweeen 0 and 1 needed to be sumed in order to exceed one. Very satisfying answer.
  • Hartogs' Extension Theorem - A study of holomorphic functions in several complex variables.
  • Conformal Mapping Techniques - Applications of conformal mappings of Complex Brownian motion.
  • Expected Value of A Random Point in Square to it's Nearest Edge - This project finds the expected distance between a point chosen at random in a unit square to the nearest edge.
